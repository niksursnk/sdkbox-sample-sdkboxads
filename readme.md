
# sdkbox Ads samples

## Getting Started

Make sure to run setup script first to download the cocos2d-x engine
~~~bash
./setup
~~~

If the setup script runs correctly the folder structure should look like this

~~~
+-- sdkbox-cocos2d-x-binary
+-- sdkbox-sample-iap
| +-- cpp
| +-- lua
| \-- js
~~~
